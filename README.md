# Fitforyou
FitnessMe is a personalized fitness application that help user with creating fitness task and track health related data ,calorie consumptions and calorie burn.  it shows a visual representation of the user personal health and calorie data. 


1.	Features
  creating personalized profile
  creating personalized fitness goal
  creating daily , monthly or weekly excercise activities and track them.
  shows a visual representation of tasks
  shows a visual representation of calorie burned by completing different tasks.
  add daily health data and see the variations over time in graphs
  add different meals and trach calorie consumed.
  track sleep hour and show a visual outlook.

2.	Tech Stack
   Frontend: React (with Vite), MUI
   Backend: Django REST Framework
   Database: SQL Server
